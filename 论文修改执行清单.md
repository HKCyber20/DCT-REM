## 0. 修改总原则（先统一口径）

- 全文只保留一套一致的符号与运动学约定：`v_c = [v_x, v_y, v_z, \omega_x, \omega_y, \omega_z]^T`（相机/探头坐标系下刚体速度）。
- 明确区分三层结论：
  - `一般3D点特征模型`（完整6DoF耦合）；
  - `PA切片局部近似模型`（可退化为平面内主导）；
  - `工程控制实现`（分层控制与近似解耦）。
- 所有“等效”“严格解耦”“渐近稳定”仅在满足条件时表述，避免绝对化。

## 1. 一级硬错误：重写式(15)–(18)（必须先改）

## 1.1 删除现有错误链条

- 删除当前由错误式(16)引出的整段推导（至少覆盖现有式(15)–(18)相关叙述）。
- 明确弃用 `\dot v_c` 写法（除非单独建立二阶模型），本节统一为速度伺服一阶模型。

## 1.2 用一致的点速度关系重建

建议替换为如下标准形式（符号可按全文约定同号变体）：

```tex
\dot{\mathbf{x}}_P = \mathbf{L}_P \mathbf{v}_c,\quad
\mathbf{L}_P = \begin{bmatrix}
-\mathbf{I}_3 & [\mathbf{x}_P]_\times
\end{bmatrix}
```

其中 `\mathbf{L}_P \in \mathbb{R}^{3\times6}`，维度闭合。

若后续只讨论成像平面坐标，可再引入投影链式关系：

```tex
\dot{\mathbf{s}} = \frac{\partial \mathbf{s}}{\partial \mathbf{x}_P}\dot{\mathbf{x}}_P
= \frac{\partial \mathbf{s}}{\partial \mathbf{x}_P}\mathbf{L}_P\mathbf{v}_c
= \mathbf{L}_s\mathbf{v}_c
```

## 1.3 解释“从一般模型到简化模型”的过渡

新增一个小节（建议放在式(17)后）：

- 先给出一般 `\mathbf{L}_s(u,v)`：理论上含 `v_z,\omega_x,\omega_y` 项；
- 再声明 PA 局部切片假设下，得到简化敏感方向；
- 明确“简化是条件成立时的近似，不是一般恒等结论”。

## 2. 解决式(17)与式(25)冲突：改成“两层模型”

## 2.1 建议替换叙述

把“式(29)与式(17)、式(13)等效”改为：

- 式(17)：一般模型（完整耦合）；
- 式(25)/(29)：在局部切片与小姿态扰动条件下的降阶近似；
- 二者是“近似退化关系”，不是“无条件等效”。

## 2.2 给出可直接粘贴的文字模板

> 为避免模型层级混淆，本文将交互矩阵分为一般形式与局部近似形式。一般形式描述三维点特征对六维速度扰动的完整一阶响应；在光声切片局部平稳、面外扰动小且纹理变化缓慢的条件下，可进一步退化为仅对平面内分量显著敏感的近似模型。故式(25)/(29)应视为式(17)在特定工作域内的降阶近似，而非全域严格等效。

## 3. 面外PBVS控制：补齐 `L_out` 雅可比链条

## 3.1 必补公式

当前从 `e_out = n \times n^*` 直接跳到控制律，需补：

```tex
\dot{\mathbf{e}}_{out} = \mathbf{L}_{out}\mathbf{v}_{out}
```

并明确

- `\dot e_d \leftrightarrow v_z` 的定义与坐标系；
- `\dot n \leftrightarrow (\omega_x,\omega_y)` 的线性化前提；
- 小角度近似下 `\mathbf{L}_{out}\approx \mathbf{I}_3` 的适用范围。

## 3.2 把“通常简化为单位阵”降级为条件性陈述

建议用：

> 在姿态误差较小、法向估计噪声受限且采样周期足够短的局部工作域内，可将 `L_out` 近似为常值满秩矩阵（常取 `I_3`）用于控制律实现。该近似仅用于局部分析，不代表全局精确模型。

---

## 4. 全局块对角解耦：从“严格”改为“近似/弱耦合”

## 4.1 修改目标

- 不再写 `\partial s/\partial v_out \equiv 0` 的绝对结论；
- 改写为“在正交投影与局部组织层变化可忽略时，一阶弱耦合可近似忽略”。

## 4.2 可粘贴模板

> 本文采用分层控制结构实现工程上的近似解耦。需要强调，平面外运动可能通过切层变化影响光声强度分布，因此严格意义上的全局块对角通常不成立。本文将该耦合视作局部小扰动下的高阶项，并在实验中验证其可控性与收敛表现。

---

## 5. 自适应 `K` 闭环定义：消除循环定义与维度跳变

## 5.1 重新定义（推荐）

- 误差评估固定在 `K_max` 维：
  - `\bar e_s` 总是在固定维度上计算；
- 控制执行使用前 `K` 维：
  - `K = f(\bar e_s)` 仅决定参与控制的截断维数。

## 5.2 建议公式模板

```tex
\bar e_s = \frac{1}{K_{max}}\sum_{i=1}^{K_{max}} w_i\,(F_i-F_i^*)^2
```

```tex
K = \mathrm{clip}\!\left(
\mathrm{round}\!\big((K_{max}-K_{min})\eta(\bar e_s)+K_{min}\big),
K_{min},K_{max}\right)
```

并加一条平滑策略（避免切换抖动）：

```tex
K_t = (1-\alpha)K_{t-1} + \alpha K_t^{raw},\quad 0<\alpha\le1
```

或设置滞回阈值（`hysteresis`）。

## 6. 稳定性证明重写：降级为“局部/条件稳定”

## 6.1 不要直接宣称“严格渐近稳定”

把结论调整为：

- 在局部线性化、小误差、`L` 满秩（或有效秩满足条件）与增益有界时，得到局部稳定/局部收敛；
- 若 `K` 自适应切换，系统属于切换/混杂系统，仅给出经验稳定性或分段Lyapunov说明。

## 6.2 需要补的条件项

- `\mathrm{rank}(L_{PA})` 的条件性讨论（不能用“系数多”替代满秩证明）；
- 投影矩阵零空间导致非零不变集的可能性；
- 对称纹理、弱纹理、退化形态下的秩亏风险。

## 6.3 可粘贴结论模板

> 在局部工作域与可观测性条件满足时，闭环误差函数满足非增性，并在常见实验工况下呈现收敛趋势。考虑到交互矩阵近似与特征维数切换机制，本文将稳定性结论限定为局部/条件意义下的稳定，而非全局严格渐近稳定。

## 8. DCT鲁棒性补丁：避免光度漂移主导误差

- 对数压缩或强度标准化；
- 可选低频抑制权重（弱化全局亮度漂移影响）。

## 8.2 文字模板
